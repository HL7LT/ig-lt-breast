### Terminology

This page lists all ValueSets and CodeSystems defined or used in the Lithuanian Breast Diagnostics Implementation Guide.

#### CodeSystems

| CodeSystem | Description |
|---|---|
| [BreastQuestionnaireOptions](CodeSystem-breast-questionnaire-options.html) | Illustrative answer option codes for the five ESPBI questionnaire forms (mammogram, tomosynthesis, ultrasound, biopsy, pathology). Covers BI-RADS conclusions, ACR density, biopsy guidance types, IHC results, tumour grading, lymphovascular invasion, and more. |

#### ValueSets

| ValueSet | Description |
|---|---|
| [BiradsCategoryLtBreast](ValueSet-birads-category-lt-breast.html) | BI-RADS assessment categories 0–6 (SNOMED CT coded). Used in BiradsAssessmentLtBreast. |
| [BreastAssessmentLtBreast](ValueSet-breast-assessment-lt-breast.html) | Breast assessment service types: ultrasound, referral, screening mammography, biopsy, tomosynthesis. Used in BreastServiceRequestLtBreast. |
| [BreastBodySiteLtBreast](ValueSet-breast-body-site-lt-breast.html) | Breast body site codes: breast, left breast, right breast. |
| [BreastClockPositionLtBreast](ValueSet-breast-clock-position-lt-breast.html) | Clock-face positions 1–12 o'clock for ultrasound lesion localization. |
| [BreastFindingLtBreast](ValueSet-breast-finding-lt-breast.html) | General breast finding types: skin mass, surgical scar, radial scar, no scarring. |
| [BreastGlandDensityACRLtBreast](ValueSet-breast-gland-density-acr-lt-breast.html) | ACR breast gland density categories: almost entirely fatty, scattered fibroglandular, heterogeneously dense, extremely dense. |
| [BreastHistorySituationLtBreast](ValueSet-breast-history-situation-lt-breast.html) | Breast history situations: prior surgery, injury, disorder, malignancy, carcinoma in situ, lump, family history. |
| [BreastNippleAppearanceFindingLtBreast](ValueSet-breast-nipple-appearance-finding-lt-breast.html) | Nipple appearance findings: normal nipple, nipple retraction. |
| [BreastQuadrantLtBreast](ValueSet-breast-quadrant-lt-breast.html) | Breast quadrant codes: upper outer, upper inner, lower inner, lower outer. |
| [FindingEvolutionLtBreast](ValueSet-finding-evolution-lt-breast.html) | Finding evolution status: new, increased, decreased, no change, absent. Used in mammographic findings. |
| [MammographicFindingLtBreast](ValueSet-mammographic-finding-lt-breast.html) | Mammographic finding types: mass, calcification, architectural distortion, focal asymmetry. |
| [UltrasoundLesionFindingLtBreast](ValueSet-ultrasound-lesion-finding-lt-breast.html) | Ultrasound lesion types: cyst, lymph node, fibroadenoma, papilloma, lipoma, malignancy, abscess. |
