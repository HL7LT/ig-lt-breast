### Terminologija

Šiame puslapyje pateikiami visi reikšmių rinkiniai ir kodų sistemos, apibrėžti arba naudojami Lietuvos krūties diagnostikos įgyvendinimo vadove.

#### Kodų sistemos

| Kodų sistema | Aprašymas |
|---|---|
| [BreastQuestionnaireOptions](CodeSystem-breast-questionnaire-options.html) | Iliustraciniai atsakymų variantų kodai, skirti penkioms ESPBI klausimynų formoms (mamografija, tomosintezė, ultragarsas, biopsija, patologija). Apima BI-RADS išvadas, ACR tankio kategorijas, biopsijos valdymo tipus, IHC rezultatus, naviko laipsnį, limfovaskulinę invaziją ir kt. |

#### Reikšmių rinkiniai

| Reikšmių rinkinys | Aprašymas |
|---|---|
| [BiradsCategoryLtBreast](ValueSet-birads-category-lt-breast.html) | BI-RADS vertinimo kategorijos 0–6 (koduotos SNOMED CT). Naudojamos profilyje BiradsAssessmentLtBreast. |
| [BreastAssessmentLtBreast](ValueSet-breast-assessment-lt-breast.html) | Krūties diagnostikos paslaugų tipai: ultragarsas, siuntimas, atrankinė mamografija, biopsija, tomosintezė. Naudojami profilyje BreastServiceRequestLtBreast. |
| [BreastBodySiteLtBreast](ValueSet-breast-body-site-lt-breast.html) | Krūties anatominių vietų kodai: krūtis, kairioji krūtis, dešinioji krūtis. |
| [BreastClockPositionLtBreast](ValueSet-breast-clock-position-lt-breast.html) | Laikrodžio padėtys nuo 1 iki 12 valandos, naudojamos ultragarsinio tyrimo pažeidinio lokalizacijai. |
| [BreastFindingLtBreast](ValueSet-breast-finding-lt-breast.html) | Bendrieji krūties radinių tipai: odos darinys, chirurginis randas, radialinis randas, randų nėra. |
| [BreastGlandDensityACRLtBreast](ValueSet-breast-gland-density-acr-lt-breast.html) | ACR krūties liaukinio audinio tankio kategorijos: beveik visiškai riebalinė, išsklaidytas fibroliaukinis audinys, nevienodai tanki, itin tanki. |
| [BreastHistorySituationLtBreast](ValueSet-breast-history-situation-lt-breast.html) | Krūties anamnezės situacijos: ankstesnė operacija, trauma, sutrikimas, piktybinis navikas, karcinoma in situ, darinys, šeimos anamnezė. |
| [BreastNippleAppearanceFindingLtBreast](ValueSet-breast-nipple-appearance-finding-lt-breast.html) | Spenelio išvaizdos radiniai: normalus spenelis, spenelio įtraukimas. |
| [BreastQuadrantLtBreast](ValueSet-breast-quadrant-lt-breast.html) | Krūties kvadrantų kodai: viršutinis išorinis, viršutinis vidinis, apatinis vidinis, apatinis išorinis. |
| [FindingEvolutionLtBreast](ValueSet-finding-evolution-lt-breast.html) | Radinio kitimo būklė: naujas, padidėjęs, sumažėjęs, nepakitęs, išnykęs. Naudojamas mamografiniuose radiniuose. |
| [MammographicFindingLtBreast](ValueSet-mammographic-finding-lt-breast.html) | Mamografinių radinių tipai: masė, kalcifikacija, architektūrinė deformacija, židininė asimetrija. |
| [UltrasoundLesionFindingLtBreast](ValueSet-ultrasound-lesion-finding-lt-breast.html) | Ultragarsinio tyrimo pažeidinių tipai: cista, limfmazgis, fibroadenoma, papiloma, lipoma, piktybinis navikas, abscesas. |
