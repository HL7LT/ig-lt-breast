### Terminologija

Šiame puslapyje pateikiami visi reikšmių rinkiniai ir kodų sistemos, apibrėžtos arba naudojamos Lietuvos krūties diagnostikos įgyvendinimo vadove.

#### Kodų sistemos

| Kodų sistema | Aprašymas |
|---|---|
| [BreastQuestionnaireOptions](CodeSystem-breast-questionnaire-options.html) | Iliustraciniai atsakymų parinkčių kodai penkioms ESPBI klausimyno formoms (mamograma, tomosintezė, ultragarsas, biopsija, patologija). Apima BI-RADS išvadas, ACR tankį, biopsijos valdymo tipus, IHC rezultatus, naviko laipsnį, limfovaskulinį invaziškumą ir kt. |

#### Reikšmių rinkiniai

| Reikšmių rinkinys | Aprašymas |
|---|---|
| [BiradsCategoryLtBreast](ValueSet-birads-category-lt-breast.html) | BI-RADS vertinimo kategorijos 0–6 (SNOMED CT koduotos). Naudojamos BiradsAssessmentLtBreast. |
| [BreastAssessmentLtBreast](ValueSet-breast-assessment-lt-breast.html) | Krūties vertinimo paslaugų tipai: ultragarsas, nukreipimas, atrankos mamografija, biopsija, tomosintezė. Naudojami BreastServiceRequestLtBreast. |
| [BreastBodySiteLtBreast](ValueSet-breast-body-site-lt-breast.html) | Krūties kūno vietos kodai: krūtis, kairioji krūtis, dešinioji krūtis. |
| [BreastClockPositionLtBreast](ValueSet-breast-clock-position-lt-breast.html) | Laikrodžio rodo padėtys 1–12 valandų ultragarsinio tyrimo pažeidimų lokalizacijai. |
| [BreastFindingLtBreast](ValueSet-breast-finding-lt-breast.html) | Bendrieji krūties radinių tipai: odos masė, chirurginis randas, radialinis randas, randų nėra. |
| [BreastGlandDensityACRLtBreast](ValueSet-breast-gland-density-acr-lt-breast.html) | ACR krūties liaukos tankio kategorijos: beveik visiškai riebalinė, išsklaidyta fibroglandulinė, nevienodai tanki, itin tanki. |
| [BreastHistorySituationLtBreast](ValueSet-breast-history-situation-lt-breast.html) | Krūties anamnezės situacijos: ankstesnė operacija, trauma, sutrikimas, piktybinis navikas, vietinis karcinoma, išaugimas, šeimos anamnezė. |
| [BreastNippleAppearanceFindingLtBreast](ValueSet-breast-nipple-appearance-finding-lt-breast.html) | Spenelio išvaizdos radiniai: normalus spenelis, spenelio įtraukimas. |
| [BreastQuadrantLtBreast](ValueSet-breast-quadrant-lt-breast.html) | Krūties kvadrantų kodai: viršutinis išorinis, viršutinis vidinis, apatinis vidinis, apatinis išorinis. |
| [FindingEvolutionLtBreast](ValueSet-finding-evolution-lt-breast.html) | Radinio evoliucijos statusas: naujas, padidėjęs, sumažėjęs, nepakitęs, išnykęs. Naudojamas mamografiniuose radiniuose. |
| [MammographicFindingLtBreast](ValueSet-mammographic-finding-lt-breast.html) | Mamografinių radinių tipai: masė, kalcifikacija, architektūrinė deformacija, židininė asimetrija. |
| [UltrasoundLesionFindingLtBreast](ValueSet-ultrasound-lesion-finding-lt-breast.html) | Ultragarsinio tyrimo pažeidimų tipai: cista, limfmazgis, fibroadenoma, papiloma, lipoma, piktybinis navikas, abscesas. |
